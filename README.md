# fleet-adapter

Launch fleet-adapter.  This step adds new elements at the visualization that is started with rmf-core.

## Pasos a seguir

```bash
git clone https://github.com/RobotnikIstobalAI2/fleet-adapter.git
cd fleet-adapter
git checkout humble-devel
cd docker
docker compose up
```
